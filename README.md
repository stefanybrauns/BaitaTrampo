# BaitaTrampo
Esse projeto tem como objetivo auxiliar a empresa BaitaTrampo na criação de um chatbot pra whatsapp, a fim de automatizar as respostas e agilizar atendimentos

Nesse fluxo abaixo contém a movimentação de mensagens e como as plataformas que utilizaremos, sejam elas n8n ou make, se conectam e ajudam na automação das mensagens e leem as planilhas.
